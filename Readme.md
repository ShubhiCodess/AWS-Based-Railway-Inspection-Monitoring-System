# AWS Cloud Pipeline for Railway Crack Detection

## Architecture

ESP32-CAM → S3 → Lambda → EC2 YOLO Inference → Dashboard

## Features
- Event-driven processing
- Cloud inference
- Scalable architecture
- Geotagged inspection logging
- Real-time monitoring

## AWS Services Used
- Amazon S3
- AWS Lambda
- EC2
- API Gateway
- CloudWatch

## Future Improvements
- GPU inference
- Multi-camera support
- Live streaming
- Edge-cloud hybrid deployment
